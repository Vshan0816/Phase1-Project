Melty Blood: Type Lumina is an fighting game based off Type Moon's visual novel Tsukihime: A Piece of Blue Glass Moon. Just like the original series, it is being developed by French-Bread. Type Lumina was released worldwide on September 30th, 2021 for Playstation 4/5, Xbox One/Series, Nintendo Switch, and PC.

Legend

The goal of this app, is to access a characters move list and frame data from Melty Blood Type Lumina

  • There are 4 inputs/buttons in the game, A,B,C, and D
  • The moveset notation is numpad notation, for example 236 corresponds to down, down-forward, forward
  • J refers to jump
  • X refers to either A/B/C, XX and XXX refer to the autocombo
  • The game runs at 60 frames per second
  • startup refers to how long it takes a move to become active
  • Frame advantage refers to how fast you can act after your move is blocked
  • If you have positive frame advantage, you act that many frames quicker
  • If you have negative frame advantage, you act that many frames slower

MoveSearch

  • The move names are prefixed by lowercase abbreviated character names as follows: shiki, arcueid, akiha, ciel, hisui, kohaku, miyako, kouma, noel, roa, vlov, redArc, saber

  • "Hold" is written after move name to indicate holding down the button

  • BC is equivalent to pressing B plus C at the same time

  • Vlov has the suffix Ice and Fire after certain special moves which have different properties
